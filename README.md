Site de Exemplo para venda de uma Landing Page para alguma Locadora de Brinquedos, em busca de oportunidade como Freelancer.

Nele contém uma Página inicial com destaque visual (hero section), Apresentação dos serviços da locadora, Seção de benefícios e diferenciais, Avaliações e Área de contato.

Este utiliza HTML5, CSS3, JavaScript, Design responsivo e uma Estrutura otimizada para landing pages.
